Technical Infrastructure Data

Detailed breakdown of each country's blockchain network footprint — Bitcoin and Ethereum node counts, mining hashrate share, and Lightning Network / validator participation. This raw data powers the Technical Infrastructure dimension of our composite index.

Technical Infrastructure Score

Five continuous pillars measured per 100k internet users, aggregated via weighted geometric mean (v2). Near-zero performance on any pillar pulls the full score toward zero.

Scoring Formula — Version 2 (Weighted Geometric Mean, Per-Capita Adjusted)

Technical Score = BTC0.25 × ETH0.25 × Mining0.15 × LN0.20 × Enabling0.15
Each pillar (except Enabling) = min(1, ln(nodes_per_100k_internet_users + 1) / ln(ceiling + 1))
BTC Nodes (w=25%): Reachable nodes per 100k internet users — log-scaled, ceiling 1.1. Source: Bitnodes.io
ETH Nodes (w=25%): Reachable nodes per 100k internet users — log-scaled, ceiling 1.2. Source: Ethernodes.org
Mining Hashrate (w=15%): % of global BTC hashrate — log-scaled, ceiling 40%. Source: Hashrate Index 2026
Lightning Network (w=20%): LN nodes per 100k internet users — log-scaled, ceiling 1.5. Source: mempool.emzy.de
Enabling Environment (w=15%): Composite of internet penetration (40%), electricity access (30%), avg speed (30%). Sources: ITU, World Bank, Speedtest
Why geometric mean? Non-compensatory: near-zero performance on any pillar pulls the composite toward zero. Recommended by OECD, Chainalysis, and HDI methodology.
Why internet users as denominator? Raw population penalizes low-connectivity countries twice. Internet users is the actual relevant base (per Chainalysis Global Adoption Index).

Data Sources

  • Bitcoin nodes: Bitnodes.io — reachable global nodes; per-100k normalized by internet user count
  • Ethereum nodes: Ethernodes.org — reachable ETH clients globally; per-100k normalized
  • Mining hashrate: Hashrate Index 2026 (US 37.5%, Russia 16.4%, Ethiopia 2.6%, Indonesia 2.1%)
  • Lightning Network: mempool.emzy.de — LN nodes per country, per-100k normalized
  • Internet users (denominator): World Population Review / ITU 2024–2025
  • Internet penetration & speed: ITU / Speedtest Global Index 2025
  • Electricity access: World Bank / Tracking SDG7 Report
  • Methodology basis: OECD Handbook on Composite Indicators (geometric mean); Chainalysis Global Adoption Index (internet-user denominator)
  • v2 key findings: India drops to Nascent (0.060) — 759M internet users makes per-capita node density tiny. UK is Advanced not a leader — near-zero mining pulls geometric mean down. Ethiopia drops from Emerging to Nascent: 2.6% mining cannot compensate for near-zero nodes.
